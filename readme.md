# Learning Manager
The idea for an app came from my technique of deciding what to study next when I'm done learning something. The user can create tasks and edit them as they progress trough.
## Technologies Used
*   Php
*   Symfony
*   Docker
*   Postgres
*   Twig
*   Tailwindcss
